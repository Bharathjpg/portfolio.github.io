# Portfolio Website

This repository hosts my personal portfolio website. Below are instructions for setting up and running the website locally, as well as deploying it.

## Prerequisites
- A modern web browser
- [Node.js](https://nodejs.org/) (for building and running the project)

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Bharathjpg/portfolio.github.io.git
   cd portfolio.github.io
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the project**:
   ```bash
   npm start
   ```
   Open your browser and navigate to `http://localhost:3000` to see the portfolio.

## Deployment
To deploy the website to GitHub Pages:
1. Push changes to the `gh-pages` branch:
   ```bash
   git push origin gh-pages
   ```
2. Visit `https://Bharathjpg.github.io/portfolio.github.io` to access your live site.
